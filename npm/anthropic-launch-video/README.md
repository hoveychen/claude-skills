# @hoveychen/anthropic-launch-video

Installer for the `anthropic-launch-video` [Claude Code](https://claude.com/claude-code) skill — a playbook for producing Anthropic-style product launch / intro videos in [Remotion](https://www.remotion.dev/) (cream + coral + Fraunces serif + JetBrains Mono terminal aesthetic), with iron-rule discipline for Player vs mp4, layer validation, reference-first frame extraction, and ffmpeg self-check.

## Install

```sh
npx @hoveychen/anthropic-launch-video install-claude-skill
```

Drops the skill into `~/.claude/skills/anthropic-launch-video/`. Re-run to update.

## Alternative: Claude Code plugin marketplace

```
/plugin marketplace add hoveychen/claude-skills
/plugin install anthropic-launch-video@hoveychen-claude-skills
```

Both routes write to the same destination — pick one.
